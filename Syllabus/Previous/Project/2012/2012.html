2012 足肢式全域地形運輸器
Terrain Walking Transporter

說明：足肢式行走器對地形的平整度要求小，經由適當設計後，可延伸至全域複雜地形之使用，本專題擬設計一足肢式行走器，不但能克服凹凸地面，在草地及大小不等之石塊存在的平台，亦能平穩的達到運輸物品的功能。

規則：所設計之行走器必須為足肢式，不得有輪子，且必須有能力行走於自強校區機械與化工系館間之草坪，行走過程不得有外力協助，並能運輸物品。行走器大小亦不得超過‭ ‬長310mm‭ ‬寬228mm‭ ‬高103cm

設計限制：
  1. 待運送之物品由助教放置於合理處，組員不得對所運送之物品做任何的改變或包覆。
  2. 動力來源為所給定之直流馬達，不得外加其他動力
  3. 必須使用下列元件至少兩種：彈簧，(正，斜，傘)齒輪，螺絲(帽)，軸承，齒條，凸輪，棘輪，皮帶，鏈條，卡榫，鉚釘。
  4. 除直流馬達外，其餘成本不得大於新台幣五千元（含所有材料成本及黏著劑等耗材）
  5. 所有加工僅能使用機械工廠設備，除同組組員外，不得假手他人
